# PhAI - Olex2 Plugin

An AI-powered plugin for [Olex2](https://www.olexsys.org/olex2/), one of the
most widely used crystallography software packages. PhAI brings machine
learning assistance directly into the structure solution and refinement
workflow, helping crystallographers make faster, more confident decisions
without leaving their existing tools.
